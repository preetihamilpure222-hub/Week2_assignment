Overview
This repository contains Python assignments completed as part of the Skill Nexis internship/training program. The programs demonstrate the use of dictionaries, file handling, JSON, CSV, and basic CRUD (Create, Read, Update, Delete) operations.

Projects Included
1. Contact Book Using Dictionary
Features:

- Add a new contact
- Search for a contact
- Update contact details
- Delete a contact
- Display all contacts

File:
"contact_book.py"

2. Word Counter from Text File
Reads a text file and displays:

- Total number of words
- Total number of lines
- Total number of characters

Files:
- "word_counter.py"
- "sample.txt"

3. JSON File Reader
Reads data from a JSON file and prints the formatted output.

Files:
- "json_reader.py"
- "student.json"

4. Student Management System (Mini Project)
A simple student management system that stores data in a CSV file.

Features:
- Add student
- Search student
- Delete student
- Save data permanently in a CSV file

Files:
- "student_management.py"
- "students.csv" (created automatically)

Technologies Used
- Python 3
- Dictionary
- File Handling
- JSON Module
- CSV Module

How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder in VS Code or any Python IDE.
4. Run the required Python file using:

python filename.py

Example:
python contact_book.py
